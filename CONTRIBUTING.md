# Contributing to xoegit

Thank you for your interest in contributing!

**First-time contributors are very welcome!** This is a beginner-friendly project — feel free to ask questions.

## Quick Start

1. **Fork & clone** the repository
2. **Install dependencies**: `npm install`
3. **Make your changes**
4. **Test**: `npm test`
5. **Submit a PR**

## Development

```bash
npm install       # Install dependencies
npm run build     # Build the project
npm test          # Run tests
npm run lint      # Check code style
npm run format    # Format code
```

## Guidelines

- Follow [Conventional Commits](https://www.conventionalcommits.org/) for commit messages
- Run `npm run lint` and `npm test` before submitting
- Keep PRs focused — one feature or fix per PR

> **Pro tip:** Use `xoegit` itself to generate your commit messages!

## Questions?

Open an issue — happy to help!
